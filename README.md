# Student_Performance_Prediction
This project builds and evaluates multiple linear regression models to analyze factors affecting students' academic performance.

## Course
This project was completed as part of Applied Mathematics and Statistics for Information Technology

## Objective
- Identify key factors influencing academic performance
- Improve prediction accuracy using feature engineering
- Evaluate model generalization with cross-validation

## Methodology
- Data preprocessing and exploratory data analysis (EDA)
- Multiple Linear Regression (baseline model)
- Feature selection and non-linear feature augmentation
- 5-fold Cross-Validation
- Evaluation metric: Mean Absolute Error (MAE)

## Results
- Best MAE: **1.595**
- Feature engineering improved model performance compared to baseline regression.
